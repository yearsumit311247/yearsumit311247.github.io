# Usability

**English (Source 1 - ISO 9241-11)**  
“Usability is the extent to which a system, product or service can be used by specified users to achieve specified goals with effectiveness, efficiency and satisfaction in a specified context of use.”

**English (Source 2 - Nielsen Norman Group)**  
“Usability is a quality attribute that assesses how easy user interfaces are to use.”

**Thai**  
Usability คือ คุณลักษณะด้านคุณภาพของระบบที่บ่งบอกถึงความง่ายในการใช้งาน โดยผู้ใช้สามารถเรียนรู้การใช้งานได้อย่างรวดเร็ว ใช้งานได้อย่างมีประสิทธิภาพ ลดข้อผิดพลาดในการใช้งาน และเกิดความพึงพอใจในการใช้งานระบบ ซึ่งเป็นปัจจัยสำคัญในการออกแบบซอฟต์แวร์ เว็บไซต์ และแอปพลิเคชัน

> "Usability คือ ความสามารถของระบบในการใช้งานได้ง่าย ผู้ใช้สามารถเรียนรู้และใช้งานได้อย่างมีประสิทธิภาพ พร้อมทั้งได้รับประสบการณ์การใช้งานที่ดี" (ChatGPT)

> "Usability คือ คุณสมบัติของระบบที่ทำให้ผู้ใช้สามารถใช้งานได้ง่าย เข้าใจได้รวดเร็ว และสามารถบรรลุเป้าหมายในการใช้งานได้อย่างมีประสิทธิภาพ" (Google Gemini)
